# Nova Digital — Brașov

Digital marketing agency landing page built with React, Vite, Tailwind CSS v4, Framer Motion, and Lucide icons.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Production Build

```bash
npm run build
npm run preview
```

## Tech Stack

- **React 19** + TypeScript
- **Vite 6** — build tool
- **Tailwind CSS v4** — utility-first CSS (compiled via `@tailwindcss/vite`)
- **Framer Motion** — animations
- **Lucide React** — SVG icons

## Features

- Brutalist dark-mode design with red accents
- Smooth scroll navigation
- Animated hero section with scroll indicator
- Interactive services accordion
- Portfolio showcase with grayscale-to-color hover
- FAQ accordion
- Contact form
- Fully responsive (375px → 1440px+)
- Accessibility: skip-link, ARIA attributes, reduced-motion support, keyboard navigation
- SEO: meta description, canonical URL, semantic HTML
